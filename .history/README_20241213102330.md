システム概要
主要機能
LINE WORKS固定メニュー
「ありがとうの木投稿」を固定メニューに追加。
選択するとWOFFで提供するウェブページに遷移します。
WOFFページ
タブ切り替え機能:
宛名、日付、内容を入力して送信するフォーム。
自分の送信履歴。
自分宛の受信履歴。
他のユーザーの一覧投稿。
Googleスプレッドシート連携
投稿内容記録。
期間ごとに送信・受信を集計。
テクノロジースタック
フロントエンド: HTML、CSS、JavaScript (バニラ)
バックエンド: Node.js (Express)
データベース : Google スプレッドシート API
デプロイ: Docker + AWS Lambda (または Heroku/GCP)
